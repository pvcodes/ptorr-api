PTORR-API

PTORR-API is a Open Sourced RESTful API for torrents. It is developed using Python.

Repository | Support Server | Docs

METHODS

► Get Torr

It will return the JSON Object consisting of torr-id and torr-title

GET: https://ptorr-api.herokuapp.com/key=batman

Required Param: /key=keyword

Optional Param: &maxCount=count(integer)

► Get New Torr

It will return the torr data of new & recent torrs

GET: https://ptorr-api.herokuapp.com/recent

► Download Torr

It will return the torr-file to be downloaded.

GET: https://ptorr-api.herokuapp.com/getorr=torr-id
